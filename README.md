# Python Calculator

Простой учебный проект калькулятора на Python, созданный для освоения Git и GitHub.

## Функциональность

*   Сложение (`add`)
*   Вычитание (`subtract`)
*   Умножение (`multiply`)
*   Деление (`divide`)
*   Возведение в степень (`power`)

## Запуск тестов

Чтобы запустить тесты, выполните команду:
```bash
pytest test_calculator.py
```
